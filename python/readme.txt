PiRoCon Example Software for Python
===================================

motorTest.py	Moves the motors forward, reverse, spin left and spin right in turn
motorTest2.py	Use the arrow keys to control motors and , and . keys to adjust speed
irTest.py	Prints the current state of IR sensors attached to pins 7, 11, 12, 13
sonarTest.py	Prints the distance in cm to nearest obstacle using ultrasonic
servoTest.py	Use the arrow keys to move pan and tilt servos, Space key to centre

initio.py	This is the library module for the Pirocon running on Initio chassis
servod		This is the software used to control the servos from within the library
